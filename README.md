# QazLatynConverter.Swift

## Usage
> ``` Swift
>
> let cyrlText = "Латын графикасына негізделген қазақ әліпбиінің конвертері"
> let latynText = QazLatynHelper.Cyrl2Latyn(cyrlText: cyrlText)
> print(latynText)
> ```
